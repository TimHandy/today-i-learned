Today I Learned

This is a simple React app, using functional stateless dumb components and smart class-based components with state.

The App pulls data from the Today I Learned Google form which saves data to a spreadsheet. The data is retrieved via a fetch command, pushed into state, and the child components are rendered via the state being passed in to the children as props.

The data is displayed in reverse date order, newest at the top.
